Modern **Inventory Management Dashboard** built with **React, Vite, and Material UI**.

This project provides a clean and responsive interface to manage:

* Products
* Orders
* Customers
* Revenue analytics
* Business growth
* Reports

Designed to be **fast, responsive, and scalable** for modern web applications.

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/MKhumaedi/Dashboard-Inventory.git
```

Go to project folder

```bash
cd Dashboard-Inventory
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open in browser

```
http://localhost:5173
```
